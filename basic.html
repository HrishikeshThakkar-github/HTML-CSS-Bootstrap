HTML Guide

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML defines the structure of a web page using a series of elements (tags) that tell the browser how to display content such as text, images, links, and other media.

Structure of HTML Page

An HTML document consists of the following structure:

  • DOCTYPE declaration: Tells the browser which version of HTML to expect. Example: <!DOCTYPE html>
  • HTML element: The root element that contains all other elements. Example: <html> ... </html>
  • Head section: Contains metadata about the document, including title and links to stylesheets or scripts. Example: <head> ... </head>
  • Body section: Contains the visible content of the page. Example: <body> ... </body>

Example:

            <!DOCTYPE html>
            <html>
                <head>
                    <title>My Web Page</title>
                </head>
                <body>
                    <h1>Welcome to My Web Page</h1>
                    <p>This is a paragraph</p>
                </body>
            </html>

HTML Elements

Introduction

HTML elements are the building blocks of an HTML page. An element typically consists of an opening tag, content, and a closing tag. The tags describe the type of content contained within them and instruct the browser on how to display it.

Elements can be simple (like a paragraph) or complex (like a table or form). In this section, we will discuss the structure, types, and common examples of HTML elements.

Basic Structure of an HTML Element

An HTML element has the following structure:

  • Opening Tag: The tag that begins the element, e.g., <h1> or <p>.
  • Content: The data or content that is enclosed by the element's tags, e.g., text or images.
  • Closing Tag: The tag that ends the element, which is usually the same as the opening tag but with a forward slash, e.g., </h1> or </p>.

Example of a basic element:

            <p>This is a paragraph.</p>

In the above example:

  • <p> is the opening tag.
  • "This is a paragraph." is the content.
  • </p> is the closing tag.

Types of HTML Elements

HTML elements can be categorized into two main types:

  • Block-level Elements: These elements typically start on a new line and take up the full width available. Examples include <div>, <h1>, <p>, and <ul>.
  • Inline Elements: These elements do not start on a new line and only take up as much width as necessary. Examples include <span>, <a>, and <img>.

Example of block-level elements:

            <h1>This is a heading</h1>
            <p>This is a paragraph of text.</p>

Example of inline elements:

            <a href="https://www.example.com">This is a link</a>
            <span>This is some inline text.</span>

Common HTML Elements

Here are some of the most commonly used HTML elements:

  • <p>: Represents a paragraph of text.
  • <h1> to <h6>: Represent different levels of headings, with <h1> being the highest and <h6> the lowest.
  • <a>: Represents a hyperlink to another webpage or resource.
  • <img>: Represents an image embedded into the page. It uses the src attribute to define the image file.
  • <ul>: Represents an unordered list, usually containing list items (<li>).
  • <ol>: Represents an ordered list, also containing list items (<li>).
  • <div>: Represents a block-level container that can group other elements.

Example of common elements:

            <p>This is a paragraph</p>
            <h1>This is a heading</h1>
            <a href="https://www.example.com">Visit Example</a>
            <img src="image.jpg" alt="Example Image">

Empty Elements

Some HTML elements do not have closing tags. These are known as self-closing or empty elements. These elements perform a specific task or represent something but do not contain content.

  • <br>: Represents a line break, often used to start a new line in text.
  • <img>: Represents an image (has a src attribute for the image URL and alt attribute for alternative text).
  • <meta>: Provides metadata about the HTML document, such as the character set or description of the page.

Example of empty elements:

            <br>
            <img src="logo.png" alt="Logo">

Nesting of Elements

Elements can be nested inside one another to create complex structures. Proper nesting ensures that the content is displayed correctly in the browser.

Example of nesting elements:

            <div>
                <h2>Welcome to My Website</h2>
                <p>This is a paragraph inside a div element.</p>
                <a href="https://www.example.com">Visit Example</a>
            </div>

Conclusion

HTML elements are the core components of any webpage. Understanding the different types of elements, their structure, and how to use them correctly will help you build well-structured and functional web pages. As you become familiar with more HTML elements, you will be able to create increasingly complex and interactive websites.
